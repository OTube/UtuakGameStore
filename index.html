Справка

		MCheckers(р. Mini Checkers) - всеми полюбившеяся настольная игра в шашки с полем 4x8. Дизайн минималистичен, используются яркие цвета и векторная крафика. Внешний вид радует глаз, но всё внимание сосредоточено на центр доски.
		Язык программирования: Java
		Quadrello - простая в оформлении, но очень захватывающая онлайн стратегия. Четыре игрока оказываются на площади 16x16, полной камней, воды и растительности. Задача каждого из них: переставлять блоки и подбирать оружие, чтобы остаться в живых.
		Язык программирования: C/C++
		Neureft - нейросеть, способная распознавать цифры с человеческой точностью.
		Язык программирования: Java, C++
		Exolupion - эволюция построенная на основе генетического алгоритма. У каждого организма есть гены, но далеко не каждый из них добъется успеха. Выживаемость может возрасти в десятки тысяч раз с начального показателя.
		Язык программирования: C/C++
		ULangPX(ulang) - интеропретируемый язык программирования для быстрой разработки в сфере научных эксперементов. Кодом легко поделиться. Производительнось-главное приемущество над другими интеропретируемыми языками.
		Язык программирования: C
